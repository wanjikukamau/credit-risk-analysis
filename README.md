# Credit Risk Analysis of a Loan Portfolio

## Project Overview

This project demonstrates a practical credit risk analysis using Microsoft Excel. The objective was to assess the health of a loan portfolio by identifying non-performing loans, measuring portfolio risk, and generating business recommendations based on the findings.

The project simulates common analytical tasks performed by investment analysts, credit analysts, and business analysts in financial institutions.

---

## Business Problem

Financial institutions need to identify high-risk loans early in order to reduce defaults and improve portfolio performance.

Using a sample loan portfolio, this project answers the following questions:

- How many loans are performing?
- How many loans are non-performing?
- What is the Non-Performing Loan (NPL) Ratio?
- What factors contribute most to loan default?
- What actions should management take?

## Key Findings

- Total Loans: **15**
- Non-Performing Loans: **3**
- NPL Ratio: **20%**

The analysis found that all non-performing loans originated from delinquent accounts, indicating that delinquency was the strongest predictor of poor portfolio performance.

---

## Recommendations

Based on the findings, the following recommendations were made:

- Monitor delinquent accounts more closely.
- Prioritize collection efforts.
- Introduce early intervention strategies.
- Track NPL trends over time.
- Continue improving portfolio monitoring.
